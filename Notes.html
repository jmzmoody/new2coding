General Notes

What is the web?

It's a bunch of computers referred to as servers that host files. The web was actually invented in the early 90's.

How do you communicate with the web?

Using a browser on your computer, you send messages using the primary protocl HTTP.

How do you create code?

In order to create code you need a special editor, Atom is the editor that I chose. One of the neat features about Atom is that you can preview your code in the editor.

An important concept to remember is that computers interpret language literally, unlike humans who often use context. A simple syntax mistake can cause large and confusing problems in code.

Basic HTML Elements

Most elements have opening and closing tags. Tags can be used to format text in bold or italics.

In addition to basic tags, HTML also utilizes attributes which provide additional information to the browser. With attributes you can insert helpful links such as this one. It provided us with an example of how our first project might look.

A few Tags that we have learned

"b" is used to bold text
"em" is used to emphasize textin italics
"br" is a void tag, that does not require an end tag and is used to provide a line break
"div" creates a "box" container element
"span" creates an "inline" container element
Void tags have no content
Inline elements are just text
Block elements create a block around text that have both heighth and width

Lesson 2: Adding Style and Flair

General Notes

HTML is a good language to begin learning code because it is so visual and the feedback can be immediate. HTML, CSS and Javascript are all languages that have different purposes in Web Development. HTML provides the structure, while CSS controls the style.

Developer Tools

In the video that we watched, we learned how to access developer tools using Google Chrome. It was interesting to find that not all elements are visible, that everything on Web page is really a square, whether it looks like one or not and that when you dig deep enough into the code, you really can find all the text!

Tree Like Structure

Your code, like your family, has a tree like structure. Translating your code into it's own "family tree" will help you see the relationships between the elements.

Making Your Web Page Stylish

When you begin to make web pages, you need to design the layout in boxes. The "div" tag will help you create the boxes. The "class" will help you add style to the content in the box.
